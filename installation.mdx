MCPJam Inspector offers multiple installation methods to fit your development workflow. Choose the method that works best for your setup.

## NPX (Recommended)

The fastest way to get started is using NPX, which downloads and runs the latest version without requiring a global installation:

```bash
npx @mcpjam/inspector@latest
```

### With Custom Options

Launch with custom port:
```bash
npx @mcpjam/inspector@latest --port 4000
```

Launch with Ollama model shortcut:
```bash
npx @mcpjam/inspector@latest --ollama llama3.2
```

Connect to an MCP server directly:
```bash
# Local FastMCP STDIO example
npx @mcpjam/inspector@latest uv run fastmcp run /path/to/server.py

# Local Node.js example  
npx @mcpjam/inspector@latest npx -y /path/to/dist/index.js
```

## Docker

Run MCPJam Inspector in a containerized environment:

### Quick Start

```bash
# Run the latest version
docker run -p 3001:3001 mcpjam/mcp-inspector:latest
```

### Background Mode

```bash
# Run in the background
docker run -d -p 3001:3001 --name mcp-inspector mcpjam/mcp-inspector:latest
```

### Docker Compose

Create a `docker-compose.yml` file:

```yaml
version: '3.8'
services:
  mcp-inspector:
    image: mcpjam/mcp-inspector:latest
    ports:
      - "3001:3001"
    environment:
      - NODE_ENV=production
    restart: unless-stopped
```

Then run:
```bash
docker-compose up -d
```

The application will be available at `http://localhost:3001`.

## From Source

For development or customization, you can install and build from source:

### Prerequisites

- Node.js 20 or higher
- npm or yarn package manager

### Clone and Setup

```bash
# Clone the repository
git clone https://github.com/MCPJam/inspector.git
cd inspector

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
# Build the application
npm run build

# Start production server
npm start
```

## System Requirements

<Note>
  **Node.js Version**: MCPJam Inspector requires Node.js 20 or higher for optimal performance and compatibility.
</Note>

- **Operating System**: Windows, macOS, or Linux
- **Memory**: Minimum 512MB RAM, 1GB+ recommended
- **Disk Space**: 100MB for installation, additional space for logs and data
- **Network**: Internet connection for downloading dependencies and AI model access

## Verification

After installation, verify that MCPJam Inspector is working correctly:

1. **Check the application loads**: Navigate to `http://localhost:3001` (or your custom port)
2. **Test MCP connection**: Try connecting to a test MCP server
3. **Verify logs**: Check the console output for any error messages

## Troubleshooting

### Port Already in Use

If you encounter port conflicts:

```bash
# Use a different port
npx @mcpjam/inspector@latest --port 4000
```

### Permission Issues (Linux/macOS)

If you encounter permission errors:

```bash
# Give execute permissions
chmod +x /path/to/inspector

# Or use sudo if necessary
sudo npx @mcpjam/inspector@latest
```

### Docker Issues

If Docker containers fail to start:

```bash
# Check Docker is running
docker --version

# View container logs
docker logs mcp-inspector

# Restart container
docker restart mcp-inspector
```
